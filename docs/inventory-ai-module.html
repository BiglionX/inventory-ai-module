📋 模块简介

这是一个基于领域驱动设计(DDD)的智能进销存管理模块，集成了 Prophet 销量预测、 n8n 自动化工作流和 Dify AI 问答能力。提供完整的库存管理、智能补货建议、 数据可视化和自然语言查询功能。

📢 模块定位声明

核心理念：传统进销存只是"记账本"，而进销存AI是"智能管家"。 它利用人工智能技术，帮助你做出更聪明的库存决策，减少积压、避免缺货、降低成本。

简单来说：这是一个"会思考的库存管理系统"，不仅能记录库存， 还能预测未来、自动补货、回答你的问题。

💡 打个比方：传统进销存像是"计算器"，只能帮你算数； 进销存AI像是"财务顾问"，不仅会算数，还能告诉你什么时候该进货、进多少、为什么。

✨ 核心功能

📊 智能销量预测

基于 Facebook Prophet 模型，支持季节性分析，提供置信区间，准确率 > 85%

🤖 自动补货建议

n8n 工作流驱动的智能化补货，自动生成采购建议，支持一键审批

💬 AI 智能问答

集成 Dify 平台，支持自然语言库存查询，语义搜索，实时响应

📈 数据可视化

Recharts 图表库，预测曲线、库存健康度仪表板、仓库利用率监控

🏗️ DDD 架构

清晰的领域驱动设计，Domain/Application/Infrastructure 分层，易于维护

⚡ 高性能

Redis 缓存策略，数据库索引优化，P95 响应时间 < 250ms

🛠️ 技术栈

Next.js 14 React 18 TypeScript FastAPI Prophet PostgreSQL Supabase Redis n8n Dify Pinecone Recharts Tailwind CSS Playwright
