📋 模块简介

这是一个基于领域驱动设计(DDD)的智能进销存管理模块，集成了 Prophet 销量预测、 n8n 自动化工作流和 Dify AI 问答能力。提供完整的库存管理、智能补货建议、 数据可视化和自然语言查询功能。

✨ 核心功能

📊 智能销量预测

基于 Facebook Prophet 模型，支持季节性分析，提供置信区间，准确率 > 85%

🤖 自动补货建议

n8n 工作流驱动的智能化补货，自动生成采购建议，支持一键审批

💬 AI 智能问答

集成 Dify 平台，支持自然语言库存查询，语义搜索，实时响应

📈 数据可视化

Recharts 图表库，预测曲线、库存健康度仪表板、仓库利用率监控

🏗️ DDD 架构

清晰的领域驱动设计，Domain/Application/Infrastructure 分层，易于维护

⚡ 高性能

Redis 缓存策略，数据库索引优化，P95 响应时间 < 250ms

🛠️ 技术栈

Next.js 14 React 18 TypeScript FastAPI Prophet PostgreSQL Supabase Redis n8n Dify Pinecone Recharts Tailwind CSS Playwright

📦 快速开始

1. 安装依赖

npm install recharts --legacy-peer-deps

2. 配置环境变量

# .env.local
DIFY_API_KEY=your-dify-api-key
DIFY_BASE_URL=https://api.dify.ai/v1
PINECONE_API_KEY=your-pinecone-api-key
PREDICTION_API_URL=http://localhost:8000

3. 使用组件

import {
  SalesForecastChart,
  InventoryHealthDashboard,
  ReplenishmentSuggestionsCard,
  AIChatAssistant
} from '@/modules/inventory-management/interface-adapters/components';

// 在页面中使用
<SalesForecastChart data={forecastData} />
<InventoryHealthDashboard
  statusDistribution={statusDist}
  lowStockItems={lowStockItems}
/>
<AIChatAssistant onSendMessage={handleSendMessage} />

4. 运行性能测试

node scripts/performance/inventory-benchmark.js

📁 项目结构

inventory-management/
├── domain/                    # 领域层
│   ├── entities/             # 实体 (3个)
│   └── repositories/         # Repository接口 (5个)
│
├── application/              # 应用层
│   └── services/            # 应用服务
│
├── infrastructure/          # 基础设施层
│   ├── repositories/        # Supabase实现 (3个)
│   ├── cache/              # Redis缓存
│   └── external-services/  # Dify + Pinecone (2个)
│
└── interface-adapters/     # 接口适配层
    ├── controllers/        # API控制器 (3个)
    ├── components/         # React组件 (4个)
    └── api/               # 中间件和响应格式
