📦 快速开始

⚙️ 环境要求

  • Node.js 18+ - JavaScript运行时
  • Python 3.9+ - 用于Prophet预测模型
  • PostgreSQL 14+ - 或使用Supabase云数据库
  • Redis - 缓存（可选但推荐）

1. 安装依赖

# 安装Node.js依赖
npm install

# 安装Recharts（可视化库）
npm install recharts --legacy-peer-deps

# 安装Python依赖（用于预测服务）
pip install prophet pandas numpy

2. 配置环境变量

创建 .env.local 文件：

# Supabase 配置
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key

# Dify AI 配置
DIFY_API_KEY=app-xxxxxxxxxxxxxxxxxxxx
DIFY_BASE_URL=https://api.dify.ai/v1

# Pinecone 配置
PINECONE_API_KEY=pcsk_xxxxxxxxxxxxxxxxxxxx
PINECONE_INDEX_NAME=inventory-knowledge

# 预测服务配置
PREDICTION_API_URL=http://localhost:8000

# Redis 配置（可选）
REDIS_URL=redis://localhost:6379

# n8n 配置（可选）
N8N_WEBHOOK_URL=http://localhost:5678/webhook

⚠️ 注意：将所有 your-xxx 替换为实际的密钥，不要将 .env.local 提交到Git。

3. 启动服务

需要启动3个服务（每个服务一个终端窗口）：

终端1：Next.js主应用 

npm run dev
# 访问 http://localhost:3001

终端2：预测API服务 

cd services/prediction-api
uvicorn main:app --reload --port 8000
# 访问 http://localhost:8000/docs

终端3：n8n工作流（可选） 

n8n start
# 访问 http://localhost:5678
